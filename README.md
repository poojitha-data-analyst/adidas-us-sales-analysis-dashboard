This project analyzes Adidas US sales data using Excel and presents insights through an interactive dashboard.

📁 Dataset

Total Records: 9,648 transactions

Includes: sales, profit, product categories, retailers, regions, and time

📌 Key Metrics

Total Units Sold: 24,78,861

Total Profit: $33.2M

Average Price per Unit: $45.22

📈 Dashboard Features

Sales trends by Month, Quarter, and Year

Top 5 Profitable Cities

Product-wise Sales Distribution

Profit by Retailer

🛠 Tools Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

📷 Dashboard Preview




🔍 Key Insights

Sales peaked in Q3 with highest quarterly sales

New York generated the highest profit

Men's Apparel contributed significantly to overall sales

📎 How to Use

Download the Excel file and use slicers to interact with the dashboard.
